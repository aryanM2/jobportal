# Job Portal

A full-stack job portal built with MERN stack (MongoDB, Express, React, Node.js).

## Features

- User registration and authentication
- Job seekers can create profiles, search jobs, and apply
- Employers can post job openings and review applications

## Setup

### Backend

1. Navigate to server directory: `cd server`
2. Install dependencies: `npm install`
3. Create a `.env` file with MongoDB URI and JWT secret
4. Run the server: `npm start` or `npm run dev`

### Frontend

1. Navigate to client directory: `cd client`
2. Install dependencies: `npm install`
3. Start the app: `npm start`

## Technologies

- Frontend: React, Tailwind CSS, Axios
- Backend: Node.js, Express, MongoDB, JWT
- Authentication: JWT tokens
